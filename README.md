## Overview

FluxMind AI is a modern SaaS-style landing page that showcases an AI-driven workflow automation platform.

The project focuses on advanced frontend engineering concepts including:

* Dynamic Pricing Matrix
* Context-Preserving Responsive Components
* Native CSS Animations
* Performance-Optimized Rendering
* Responsive Bento Grid Layouts
* SEO-Friendly Architecture
* Accessible Semantic Markup

The application is built using Next.js App Router and TypeScript with a strong emphasis on maintainability, performance, and scalability.

## Engineering Highlights

### Pricing Matrix

A configurable pricing engine calculates plan costs dynamically based on:

* Base Rate
* Currency Multiplier
* Billing Cycle
* Regional Adjustments

No hardcoded UI pricing values are used.

### Context Lock

The Features section preserves the active state when switching between:

```txt
Desktop Bento Grid
        ↓
Mobile Accordion
```

ensuring a seamless responsive experience.

### Native Motion System

All interactions are implemented using:

* CSS Transitions
* CSS Keyframes
* Web Animations API (WAAPI)

No animation frameworks are used.

## Local Development

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```txt
http://localhost:3000
```

Production build:

```bash
npm run build
```

Start production server:

```bash
npm start
```
